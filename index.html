О приложении

Этот сервис заметок разработан для интеграции в Telegram Web Apps и оптимизирован под мобильные устройства.

Стек технологий

  • HTML5
  • SCSS (модерновая организация стилей с переменными и вложенностью)
  • Vanilla JavaScript (ES Modules, модульная архитектура, дебаунсы, управление состоянием)
  • Quill.js — WYSIWYG-редактор
  • Telegram Web Apps API — интеграция с темой, жестами и поведением в Telegram
  • LocalStorage — локальное хранение заметок

Подходы

  • Mobile-first дизайн, большие тач-цели
  • Адаптация темы под Telegram (светлая/тёмная, цвета темы)
  • Надёжное локальное сохранение: автосохранение, явная кнопка «Сохранить»
  • Оптимистичные UI-отклики: тосты, тактильная отдача если доступна
  • Долгое нажатие для быстрых действий (удаление заметки)
  • Модульная структура кода и доступность (ARIA-атрибуты для меню)
